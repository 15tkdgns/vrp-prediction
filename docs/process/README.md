## 🔍 의사결정 포인트

### 1. 타겟 변수 선택
- **선택**: 변동성 예측 (target_vol_5d)
- **기각**: 수익률 예측 (R² ≈ 0)
- **근거**: 효율적 시장 가설, 변동성 지속성

### 2. 모델 선택
- **선택**: Ridge Regression (alpha=1.0)
- **비교**: Lasso, ElasticNet, RandomForest, HAR
- **근거**: 안정성, 해석 가능성, HAR 대비 우수

### 3. 검증 방법
- **선택**: Purged K-Fold CV (5-fold)
- **기각**: 단순 K-Fold, TimeSeriesSplit
- **근거**: 금융 시계열 표준 (López de Prado)

### 4. 데이터 소스
- **선택**: yfinance (실제 SPY 데이터)
- **기각**: Random 시뮬레이션, 합성 데이터
- **근거**: 데이터 무결성, 재현 가능성
