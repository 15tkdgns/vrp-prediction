1. 연구 개요
================================================================================

1.1 연구 제목
-------------
머신러닝을 활용한 5일 변동성 위험 프리미엄(VRP) 예측 연구

1.2 연구 목적
-------------
1) 5일 단기 호라이즌에서 실현 변동성(RV) 예측 가능성 검증
2) VIX(내재 변동성)의 예측 기여도 분석
3) 예측 모델의 경제적 가치 입증

1.3 연구 질문 (Research Questions)
----------------------------------
RQ1: 5일 호라이즌에서 실현 변동성을 예측할 수 있는가?
RQ2: VIX(내재 변동성)가 예측에 얼마나 기여하는가?
RQ3: 예측 모델이 경제적 가치를 제공하는가?


2. 문헌 연구 및 연구 갭
================================================================================

2.1 기존 연구
-------------
- HAR-RV (Corsi, 2009): 일간/주간/월간 RV 활용
- GARCH (Bollerslev, 1986): 조건부 이분산성
- Realized GARCH (Hansen et al., 2012): RV와 GARCH 결합
- VRP 정의 (Carr & Wu, 2009): IV² - RV
- VRP 예측력 (Bollerslev et al., 2009): VRP가 수익률 예측

2.2 연구 갭
-----------
- 기존 연구는 주로 22일(월간) 호라이즌에 집중
- 단일 모델(HAR-RV 또는 GARCH) 단독 사용
- Walk-Forward CV 등 엄격한 검증 부족
- 경제적 가치 검증 미흡

2.3 본 연구의 차별점
--------------------
- 5일 단기 예측 호라이즌 (실용적)
- VIX 통합 (내재 변동성 정보 활용)
- Walk-Forward CV + Diebold-Mariano 검정
- 트레이딩 시뮬레이션으로 경제적 가치 입증

3.5 검증 방법
-------------
1) Walk-Forward Cross-Validation (5-Fold)
   - 시간 순서 유지
   - Gap = 5일 (예측 호라이즌)
   
   Fold 1: Train[0:169] → Gap[5] → Test[170:335]
   Fold 2: Train[0:335] → Gap[5] → Test[336:501]
   Fold 3: Train[0:501] → Gap[5] → Test[502:667]
   Fold 4: Train[0:667] → Gap[5] → Test[668:833]
   Fold 5: Train[0:833] → Gap[5] → Test[834:999]

2) Diebold-Mariano Test
   - Persistence 벤치마크 대비 통계적 유의성 검정

3) Bootstrap Confidence Interval
   - 100회 Bootstrap으로 R² 95% 신뢰구간 계산

---------------
✓ Walk-Forward CV R² = 0.18 (SPY, QQQ)

------------------------
✓ 주식 지수 (SPY, QQQ): R² > 0.25 → 우수

✓ 원유 (USO): R² = 0.24 → 양호
✓ 신흥시장 (EEM): R² = 0.23 → 양호
✗ 금 (GLD): R² < 0 → 예측 불가
✗ 채권 (TLT): R² < 0 → 예측 불가

7. 결론 및 시사점
================================================================================

7.1 연구 기여
-------------
1. VIX 중요성 확인: 내재 변동성이 단기 예측에 핵심
2. 타겟 변환 효과: Sqrt/Log 변환이 원시값보다 우수
3. 경제적 가치 입증: 실제 트레이딩에서 초과 수익

7.2 한계점
----------
- Pre-COVID가 Post-COVID보다 예측력 우수 (환경 변화)
- 극단적 VIX 구간에서 예측력 저하
- Train/Test 분할 시점에 따른 성능 변동
- 금, 채권 등 비주식 자산에서 예측 실패
