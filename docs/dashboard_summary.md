## 핵심 실험 결과 요약

### 실험 설정
- **자산**: SPY, GLD, TLT, QQQ, EEM (5개)
- **모델**: ElasticNet, Ridge, Lasso, RandomForest, GradientBoosting, MLP_64, MLP_128_64 (7개)
- **분할**: 80:20 (22일 Gap 적용)
- **타겟**: 22일 후 실현 변동성 (RV_22d_future)

### 자산별 최고 성능 모델
| 자산 | 최고 모델 | R2 | 방향 정확도 |
|------|----------|-----|------------|
| SPY | MLP_64 | -0.97 | 54.5% |
| GLD | Lasso | -0.20 | 38.0% |
| TLT | Lasso | -1.30 | 45.5% |
| QQQ | MLP_64 | -0.37 | 52.1% |
| EEM | MLP_128_64 | -0.88 | 58.7% |

### 모델 유형별 평균 R2
| 유형 | 모델 | 평균 R2 |
|------|------|---------|
| 선형 | ElasticNet, Ridge, Lasso | ~-0.9 |
| 신경망 | MLP_64, MLP_128_64 | ~-1.1 |
| 트리 | RandomForest, GradientBoosting | ~-2.5 |

### 주요 인사이트

1. **22일 후 RV 예측은 매우 어려운 과제**
   - 모든 모델에서 R2 < 0 (단순 평균 예측보다 낮음)
   - 22일 Gap 적용으로 현실적 예측 환경 반영

2. **자산별 예측 난이도 차이**
   - GLD (금): 가장 예측 가능 (VIX와 낮은 상관관계)
   - TLT (국채): 가장 어려움 (변동성 패턴 불규칙)

3. **선형 모델의 안정성**
   - 과적합 없이 일관된 성능
   - Lasso가 GLD, TLT에서 최고 성능

4. **신경망의 특정 자산 우위**
   - MLP_64가 SPY, QQQ에서 선형 모델 상회
   - 비선형 패턴 포착 가능

5. **트리 모델의 과적합 문제**
   - RandomForest, GradientBoosting 모두 낮은 R2
   - 시계열 데이터에 부적합
