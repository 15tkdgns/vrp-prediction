## 4. 핵심 인사이트

### 4.1 예측 난이도
- **모든 모델에서 R2 < 0**: 22일 후 RV 예측은 매우 어려운 과제
- 22일 Gap 적용으로 현실적 예측 환경 반영
- 단순 평균 예측보다 낮은 성능 → 시계열 예측의 본질적 한계

### 4.2 자산별 예측력 차이
- **GLD (금)**: 가장 예측 가능 (R2 ~ -0.2)
  - VIX와 낮은 상관관계 → 예측 가능한 오차 존재
- **TLT (국채)**: 가장 어려움 (R2 ~ -1.3 이하)
  - 금리 정책에 민감, 변동성 패턴 불규칙

### 4.3 모델 선택 가이드
- **안정성 중시**: Lasso, ElasticNet, Ridge (선형 모델)
- **주식 자산 (SPY, QQQ)**: MLP_64가 선형 모델 상회
- **피해야 할 모델**: RandomForest, GradientBoosting (과적합)

### 4.4 최종 권장 모델
- **ElasticNet** 권장
  - 해석 가능 (계수 직접 확인)
  - 안정적 성능
  - Lasso와 유사한 성능
  - 다중공선성 처리 가능
