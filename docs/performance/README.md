## 🔍 성능 검증 방법론

### 1. 데이터 누출 방지
- **PurgedKFold**: 시계열 데이터 누출 방지
- **시점 분리**: t-1 기준 엄격한 특성 생성
- **자동 감지**: 누출 의심 패턴 자동 탐지

### 2. 통계적 검증
- **교차 검증**: 5-fold 시계열 분할
- **부트스트랩**: 신뢰구간 계산
- **유의성 검정**: p-value < 0.05 기준

### 3. 실제 거래 환경 검증
- **거래 비용**: 슬리피지, 수수료, 마켓 임팩트
- **백테스팅**: 실제 거래 조건 반영
- **위험 지표**: VaR, CVaR, 최대 낙폭
