# SPY 2025 상반기 분석 도구

SPY (SPDR S&P 500 ETF Trust) 2025년 상반기 실제 가격 데이터와 AI 예측 신호를 비교 분석하는 웹 애플리케이션입니다.

## 📊 주요 기능

- **실시간 차트**: 2025년 1월~6월 SPY 실제 가격 추이
- **AI 예측 신호**: 기술적 분석 기반 상승/하락 예측 표시
- **상세 통계**: 월별 예측 정확도, 수익률, 가격 변동 분석
- **반응형 디자인**: 모바일/데스크톱 최적화

## 🚀 빠른 시작

### 사전 요구사항
- Node.js (v14 이상)
- 웹 브라우저 (Chrome, Firefox, Safari 등)

### 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 시작
npm run dev

# 브라우저에서 접속
# http://localhost:8080
```

## 🎯 사용법

1. **데이터 로딩**: 페이지 로드 시 자동으로 SPY 데이터와 예측 데이터를 불러옵니다.
2. **차트 분석**: 실제 가격 라인과 예측 신호(상승/하락)를 확인할 수 있습니다.
3. **통계 확인**: 하단 통계 카드에서 상세 분석 정보를 확인할 수 있습니다.
4. **반응형**: 모바일에서도 최적화된 환경으로 분석 가능합니다.
